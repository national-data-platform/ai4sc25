Workshop on Generative AI and FAIR Principles in Science Communication (AI4SC-2024)

Call for Papers

We invite submissions to the Workshop on Generative AI and FAIR Principles in Science Communication (AI4SC-2024), co-located with the 20th IEEE e-Science 2024. This workshop explores the intersection of generative AI, FAIR principles, and scientific communication and software. Topics include, but are not limited to:

  • Generative AI and LLM to communicate scientific innovations
  • AI-driven techniques for scientific communication and software repositories
  • Blue-sky research: Future AI for scientific software and repositories
  • Different approaches to process data, like RAG, finetuning, agentic AI
  • Novel methods for improving software discovery, usability, FAIRness, and automation
  • AI-powered software and repository similarity analysis and documentation generation
  • Constructing, deploying, and managing ontologies from domain repositories using generative AI
  • Next-generation GenAI for scientific software and repository interaction to communicate scientific methodology and technique
  • AI-powered software and repository discovery & reusability, creating better protocols for scientific communities
  • Privacy risks of scientific communication through generative AI, including access control, machine learning, and more
  • Privacy-preserving heterogeneous data repositories
  • LLM-based agentic pipelines for scientific workflows
  • Breakthroughs in LLM for scientific advancements using big data and HPC techniques
  • Integrating knowledge graphs and ontologies with generative AI for innovation and result representation

The workshop welcomes the following types of submissions:

  • Full papers (10 pages including references): Presenting novel research, systems, or case studies.
  • Posters (2 pages): Showcasing early-stage results or standalone innovations.

All submissions will be single-blind peer reviewed. Accepted full papers will be presented orally. Posters will be featured in a dedicated session. Both will appear in the e-Science 2024 proceedings. Rejected full papers may be re-submitted as posters.

At least one author per accepted submission must register for the workshop at the full rate. Each author registration covers one accepted paper/poster.
